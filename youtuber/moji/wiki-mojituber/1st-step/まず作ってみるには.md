# 簡単な台本がある前提で、まずyoutube動画を1本作るには(動画編集＋読み上げツール)

文字ノウハウ系（顔出しなし／自分で喋らない）で、**最短で1本投稿する**ための手順とテンプレをまとめる。

## 0) 結論（最短ルート）
- **原稿（テキスト）** → **読み上げで音声生成** → **編集で音声＋文字を載せる** → **書き出して投稿**
- 最初は **1テーマ1本／3〜7分（できれば5分以内）** で十分。

## 1) 最低限必要なもの（これだけで始められる）
- **原稿**：メモ帳 / Markdown（箇条書きでもOK）
- **音声**：テキスト読み上げ（TTS） or ゆっくり系音声
  - ※TTSとは＝入力した文章をAI音声で読み上げて、音声ファイルを書き出せる機能
- **編集ソフト**：無料でOK（まずは iMovie か CapCut）
- **素材（任意）**：画像・画面キャプチャ・図解（なくても成立）

※ 顔出し・撮影機材・高性能PCは必須ではない。

## 2) 動画編集ソフトについて（Mac）
### 2-1) 今回の用途（台本＋読み上げ／キャラ絵なし）で重要なこと
- 必要なのは主に **①音声を置く ②要点テロップ ③（必要なら）画像/画面キャプチャ**。
- つまり「高度なエフェクト」より **文字の入れやすさ／書き出しの安定／軽さ**が大事。

### 2-2) まず始めるなら（未経験者向け）
#### iMovie（無料・標準搭載 / Mac・iPhone）
- **良い点**：操作が直感的／軽め／音声配置＋テロップがすぐできる／最短で1本出せる。
- **悪い点**：凝ったテロップやテンプレ量産は弱め／機能がシンプル（伸びた後に物足りなくなることがある）。

#### CapCut（無料 / スマホ・Win・Mac）
- **良い点**：テロップが強い／テンプレ・プリセットで時短しやすい／TTSも使えて一体運用しやすい。
- **悪い点**：
  - 機能が多くて迷いやすい／端末や素材によっては重くなることがある。
  - 中国製アプリである
    - 中国の大手IT企業 ByteDance（バイトダンス）社が開発・提供している
    - CapCut（キャップカット）の隠された危険性と代替動画編集ソフト4選　【2025年7月10日】
      - https://stella-international.co.jp/ai/capcut-safety/?utm_source=chatgpt.com
        - Capcut（キャップカット）に潜む5つの危険性
          - １.編集した動画に対する各種権利を放棄させられる
            - つまりユーザーが作成したにもかかわらず、動画はByteDance社のものです。よって動画を勝手に使われて編集されても、文句はいえません。
            - 編集した動画に関する権利を取り上げられるのは、ユーザーの活動内容によってはかなり大きな制約になります
          - ２.年齢制限が設けられている
          - ３.商用利用が禁止されている
            - 収入目的でなく、個人の趣味で楽しむ場合だけCapCutを利用できます。
            - **裏を返すと、収益を得る目的では利用できません**
          - ４.プライバシーが守られない可能性がある
          - ５.楽曲が著作権フリーではない
        - Capcutより危険性が低い動画編集アプリ4選
          - １.iMOVIE
          - ２.PowerDirector
          - ３.Adobe Premiere Pro
          - ４.Filmora


※この点は 公式条文を必ず確認すべきであり、ここに断定的な法律解釈は示しません。ただし、少なくない情報源が「権利放棄/包括的許諾と読み取れる条項がある」と指摘しています

### 2-3) 本格的にやるなら（次のステップ）
#### DaVinci Resolve（無料版あり / Win・Mac・Linux）
- **良い点**：無料でも高機能（音声・色・書き出しが強い）／長期的に伸ばしやすい。
- **悪い点**：学習コストが高い／低スペMacだと重くなりやすい（今回の用途だとオーバースペックになりがち）。

### 2-4) 代替案（無料 / 軽め）
#### Shotcut / OpenShot（Win・Mac・Linux）
- **良い点**：無料で基本は揃う／比較的軽めの構成で済む。
- **悪い点**：日本語テロップの体験やテンプレ量産は、CapCutほど楽ではないことがある。

### 2-5) 参考：他の編集ソフト
- **Premiere Pro（Win・Mac）**：**完全無料ではない**（体験期間の後は月額）。業界標準だが最初から必須ではない。
- **AviUtl（Windowsのみ）**：ゆっくり解説文化の定番（導入・設定のクセが強め）。
- ゆっくりMovieMaker4
  - 開発者：饅頭遣い（個人）
    - 配布サイト：饅頭遣いのおもちゃ箱（manjubox.net）
    - 開発は 日本の個人／コミュニティベースです。
    - フリーソフトとして無償配布
  - ソフト概要ゆっくりMovieMaker4（略称：YMM4）
    - 種類： Windows向け 無料動画編集ソフト
    - 用途： 「ゆっくり実況／ゆっくり解説」系動画の制作支援
    - 特徴： 読み上げ音声 ＋ 編集（字幕・立ち絵・BGM・カット編集等） を一つのアプリで完結可能 
    - ※ 2025年時点では最新版「YMM4 4.46.1.x」などが配布されています
    - 基本機能（動画制作）
      - テキスト入力による音声生成（ゆっくり音声系）
      - タイムライン編集（カット・配置・重ね順）
      - 字幕表示・文字装飾
      - 立ち絵アイテム（画像・表情差分）
      - エフェクト（出現・退場・ぼかし・集中線など）
      - テンプレート機能（よく使う構成の保存）
      - 動画の出力（エンコード）
      - ※ 古い「YMM3」まで必要だった外部エンコードなしで完結可能 
      - ウィキペディア
    - 音声／読み上げ
      - 内蔵の AquesTalk系（ゆっくりボイス） を利用可能
      - ※ 商用利用時は別途ライセンスが必要になる場合あり 
      - 外部音声合成エンジンの連携が可能（CeVIO、VOICEVOX 等）
    - ゆっくりMovieMaker4とゆっくりMovieMaker4 Liteがある
      - ゆっくりMovieMaker4はAquestalkのライセンス購入（年間六千円ちょっと）が必要
      - ゆっくりMovieMaker4 Liteだと別の音声合成ソフトが必要になる
    - 結論
      - 初心者はゆっくりMovieMaker4 Lite　＋ 音声合成ソフト（VoiceVoxがおすすめ）で作れば、商用でも無料で済む
        - VoiceVoxは、基本的には動画の最後に出典元を書けば良いが、声によるので、ケースバイケース、要確認。

### 2-6) 低スペMacBook向けの注意（重くしないコツ）
- **1080p / 30fps** で十分（まずは4Kにしない）
- **素材を減らす**：画像や動画の重ねすぎ・アニメーション多用を避ける
- **テロップは「要点のみ」**（全文表示しない＝編集も軽くなる）
- 可能なら **短尺（3〜5分）** で回して、作業を分割する

### 2-7) 使える端末（ざっくり）
- **iMovie**：Mac / iPhone（※Windows・Androidは不可）
- **CapCut**：iPhone / Android / Windows / Mac
- **DaVinci Resolve**：Windows / Mac（※スマホは不可）
- **Shotcut / OpenShot**：Windows / Mac（※スマホは基本不可）
- **AviUtl**：Windowsのみ

### 2-8) スマホだけで完結したい人向け（スマホ専用の代表例）
- **InShot（iPhone/Android）**：操作が簡単で、テロップ＋音声の合成が手早い。
- **KineMaster（iPhone/Android）**：スマホ編集の定番。細かい調整もしやすい。

※ スマホ専用/提供状況はアップデートで変わるので、インストール前に対応OSを確認すると安全。

## 3) テキスト読み上げ（TTS）は無料で使える？
**使える（実用レベル）。**

- **VOICEVOX**：無料で高品質。規約に従えば商用利用も可。
- **CapCutのTTS**：手軽。編集と一体で回したい場合に便利。
- **macOS標準の読み上げ**：内蔵機能。まず試すには十分。
- **オンラインTTS（例：Natural Reader等）**：ソフト不要で手軽（商用可否・利用規約は都度確認）。

※ ナレーション収録用マイクは必須ではない（TTS運用なら不要）。

## 4) 「まりさのゆっくり解説」はどうやって作っている？
ざっくり言うと、**テキスト→合成音声→静止画＋テロップ**。

- **音声**：AquesTalk系などの合成音声（「ゆっくり霊夢」「ゆっくり魔理沙」等）
- **映像**：キャラ立ち絵（口パク差分）＋テロップ中心
- **編集**：AviUtl が長年の定番（今は他ソフトでも同様に作れる）

## 5) 超シンプル制作フロー（初心者向け）
1. **原稿を書く**（まずは箇条書きでOK）
2. **読み上げソフトで音声生成**（VOICEVOX等で書き出し）
3. **編集ソフトで音声を配置**
4. **要点だけテロップ**（全文は出さない）
5. **必要なら画像を追加**（なくてもOK）
6. **書き出してYouTubeへ投稿**

## 6) おすすめ構成（伸びやすい“型”）
- **結論 → 理由 → 具体例 → まとめ**
- **1動画 = 1テーマ**
- **テロップは全文表示しない（要点のみ）**

## 7) コピペ用：ゆっくり解説テンプレ（台本）
### 7-1) 動画コンセプト（最初に埋める）
- **テーマ**：
- **想定視聴者**：
- **視聴後に「わかるようになること」**：
- **動画時間目安**：3〜7分（最初は5分以内）

### 7-2) 登場キャラ（最小構成）
- **ゆっくり霊夢**：進行役・視聴者目線
- **ゆっくり魔理沙**：解説役・結論担当

### 7-3) 台本（そのまま使う）
**オープニング（〜20秒）**
- 【霊夢】ゆっくり霊夢です。今回は「◯◯」を解説するよ。
- 【魔理沙】結論から言うと、◯◯は△△だからだぜ。
- 【霊夢】「なんとなく難しい…」って人は最後まで見てね。

**本編①：問題提起（なぜ？）**
- 【霊夢】そもそも、なぜ◯◯は分かりにくいの？
- 【魔理沙】理由は3つ。①□□ ②△△ ③××だ。

**本編②：具体例（理解フェーズ）**
- 【霊夢】具体的にはどういうこと？
- 【魔理沙】例えば…（実例を1つ）。それが挫折の原因になりやすい。

**本編③：整理・構造化（重要）**
- 【魔理沙】整理するとこうなる。
  - **全体像**：□□
  - **よくある誤解**：△△
  - **正しい順番**：①→②→③

**まとめ（再度結論）**
- 【霊夢】今日のポイントはこの3つだね。
- 【魔理沙】（要点を3つ箇条書き）これだけ覚えればOKだ。

**エンディング（CTA）**
- 【霊夢】役に立ったら高評価・チャンネル登録お願いします。
- 【魔理沙】質問やリクエストはコメントで頼むぜ。

## 8) 編集構成テンプレ（iMovie / CapCut / DaVinci共通）
- **BGM**：小さめ（常時でもOK）
- **音声**：聞き取りやすさ優先（音量を揃える）
- **テロップ**：全文は出さない（要点のみ）
- **キャラ画像**：使うなら左右固定（切り替えだけ）

### 8-1) シーン構成（タイムライン目安：5分）
- 0:00〜0:20：オープニング
- 0:20〜1:30：問題提起
- 1:30〜3:30：解説・具体例
- 3:30〜4:30：整理・構造化
- 4:30〜5:00：まとめ・CTA

## 9) 初心者向け：やりがちなNG
- 台本が長すぎる
- 結論が最後まで出てこない
- テロップが文字だらけ（全文表示）
- キャラや演出を増やしすぎる

## 10) 初心者向け量産ルール（最重要）
- **完璧を目指さない**
- **テンプレ固定で量産**
- **改善は10本出してから**（まずはサンプルを貯める）

## 12) MacBook無料ツール：おすすめセット（目的別）
※ 目的とPC性能で選ぶ。

- **① 最速で1本作る**：iMovie（編集） + VOICEVOX（読み上げ）
- **② 1つで完結したい**：CapCut（編集＋TTS）
- **③ 本格運用に伸ばす**：DaVinci Resolve（編集） + VOICEVOX（読み上げ）

### 12-1) 結論：あなたの前提（低スペMac／台本あり／読み上げ／キャラ絵なし）なら
- **まず②（CapCut）で1本作る** → 重ければ **①（iMovie+VOICEVOX）に切り替え** → 本格化したら **③** を検討、が現実的。

### 12-2) ①〜③の使い分け（良い点・注意点）
#### ① 最も安定して軽く回したい：iMovie（編集） + VOICEVOX（読み上げ）
- **向いている**：低スペでも「編集だけ」を軽く済ませたい／音声はVOICEVOXで別途しっかり作りたい。
- **良い点**：編集が軽めで安定しやすい／作業が単純（音声→編集）／落ちにくい。
- **注意点**：テロップ表現はCapCutほど多彩ではない（要点テロップ運用に寄せると楽）。

#### ② 最速で1本出したい・迷いたくない：CapCut（編集＋TTS）
- **向いている**：とにかく最初の1〜5本を早く出して流れを掴みたい／1つのアプリで完結したい。
- **良い点**：テロップが強い／テンプレで時短しやすい／TTSも同一アプリで回せる（ワンストップ）。
- **注意点**：素材や演出を盛ると重くなりやすい（低スペMacは特に）。まずは「音声＋要点テロップ」だけで作る。

#### ③ 長期的に作り込みたい：DaVinci Resolve（編集） + VOICEVOX（読み上げ）
- **向いている**：編集や音まで追い込みたい／将来的に本格運用したい。
- **良い点**：高機能（特に音・書き出し周り）／上達すれば伸びしろが大きい。
- **注意点**：低スペMacだと重くなりがち。最初の数本で選ぶと挫折要因になりやすい。

### 12-3) 切り替え判断（迷ったらこれ）
- **②で作っていて重い／落ちる／書き出しが辛い** → **①へ**（編集を軽くする）
- **①で作れているが、もっと表現やテンプレ量産をしたい** → **②を併用**（テロップ強化）
- **継続できて、作り込みニーズが出てきた** → **③を検討**
