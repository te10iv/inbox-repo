# ダイジェスト：体のパーツ（頭からつま先まで）

体のパーツを「頭 → つま先」の順で、まとめて一気に確認するためのダイジェストです。

---

## ダイジェスト

### 頭からつま先までを単語で列挙（小学生向け）

**単語（頭→つま先）**  
head（頭） → hair（髪） → face（顔） → forehead（額） → eye / eyes（目） → ear / ears（耳） → nose（鼻） → mouth（口） → tooth / teeth（歯） → tongue（舌） → neck（首） → shoulder（肩） → arm（腕） → elbow（ひじ） → wrist（手首） → hand（手） → finger（指） → thumb（親指） → nail（爪） → chest（胸） → back（背中） → stomach（お腹） → waist（腰） → hip（腰・尻のあたり） → leg（脚） → knee（ひざ） → ankle（足首） → foot / feet（足） → heel（かかと） → toe / toes（つま先）

**例文（1行）**: Touch your head, shoulders, knees, and toes.（頭、肩、ひざ、つま先に触って。）

---

### 頭からつま先までを単語で列挙（中学〜高校向け）

**単語（頭→つま先）**  
scalp（頭皮） → skull（頭がい骨） → forehead（額） → temple（こめかみ） → eyebrow（まゆ毛） → eyelid（まぶた） → eyelash（まつ毛） → iris（虹彩） → pupil（瞳孔） → cheek（頬） → cheekbone（頬骨） → jaw（あご・顎の骨） → chin（あご先） → nostril（鼻の穴） → lips（唇） → gums（歯ぐき） → tongue（舌） → throat（のど） → Adam's apple（喉仏） → nape（うなじ） → neck（首） → collarbone（鎖骨） → shoulder（肩） → shoulder blade（肩甲骨） → armpit（わきの下） → upper arm（上腕） → biceps（二頭筋） → triceps（三頭筋） → forearm（前腕） → elbow（ひじ） → wrist（手首） → palm（手のひら） → knuckle（指の関節） → fingernail（手の爪） → ribs（肋骨） → abdomen（腹部） → navel（へそ） → spine（背骨） → lower back（腰／腰部） → buttocks（お尻） → groin（股間） → thigh（太もも） → hamstring（ハムストリング） → kneecap（膝蓋骨） → shin（すね） → calf（ふくらはぎ） → Achilles tendon（アキレス腱） → ankle（足首） → heel（かかと） → sole（足の裏） → toenail（足の爪） → toes（つま先）

**例文（1行）**: He injured his Achilles tendon, and his calf hurts.（彼はアキレス腱を痛めて、ふくらはぎが痛い。）
