# 2. VOICEVOX→iMovieで最短1本を作ります（低スペMac向け）

このページは、**VOICEVOXで音声を作り**、**iMovieで編集して**、短時間でYouTube動画を1本完成させます。あなたは低スペMacBookでも安定して回しやすい流れを選びます。

## 2-1. 結論
あなたは「音声はVOICEVOXで作る」「編集はiMovieで軽く済ませる」と、**迷いと負荷を減らして最短で1本を出せます**。

## 2-2. 全体像
### 2-2-1. 目標を定義します
- あなたは「台本読み上げ＋要点テロップ」だけで成立させます。
- あなたは「キャラ絵」「凝った演出」を入れず、編集を軽くします。

### 2-2-2. 作業の流れを確認します
1. あなたは台本を整えます。
2. あなたはVOICEVOXで音声（WAV推奨）を書き出します。
3. あなたはiMovieに音声と背景素材を読み込みます。
4. あなたは要点テロップを置きます。
5. あなたは1080p/30fps相当で書き出します。

## 2-3. 詳細（手順）
### 2-3-1. 台本を「読み上げ用」に整えます
あなたは読み間違いと不自然さを減らします。

- あなたは記号や括弧を減らします（TTSが詰まりやすいです）。
- あなたは一文を短くします（目安：1文40〜60文字）。
- あなたは数字に読みを補います（例：SAA→エスエーエー など）。
- あなたは句読点で間を作ります（息継ぎが自然になります）。

### 2-3-2. VOICEVOXで音声を作成して書き出します
1. あなたはVOICEVOXを起動します。
2. あなたは話者とスタイルを選びます。
3. あなたはテキストを入力します。
4. あなたは試聴して、違和感がある箇所だけ直します。
5. あなたは音声を**1本に繋げて書き出し**ます。

#### 書き出しのコツ
- あなたはファイル名に日付とテーマを入れます（例：`2025-12-22_aws_saa_挫折理由.wav`）。
- あなたは最初はWAVで書き出します（変換で劣化しにくいです）。

### 2-3-3. iMovieで編集します（最低限）
1. あなたはiMovieで新規プロジェクトを作成します。
2. あなたは背景（単色でも可）や画像をタイムラインに置きます。
3. あなたはVOICEVOX音声をタイムラインに置きます。
4. あなたは「タイトル（テロップ）」で要点テロップを置きます。

#### 低スペMacで重くしないコツ
- あなたは解像度は1080pを前提にします。
- あなたは素材の重ねを最小限にします（背景＋テロップ程度にします）。
- あなたはアニメーションやトランジションを増やしません。

### 2-3-4. 要点テロップを置きます（全文は出しません）
あなたは「聞きながら理解できる」ことを優先します。

1. あなたは台本から要点を抜きます。
2. あなたは1画面1メッセージにします。
3. あなたは表示時間を音声に合わせます。

### 2-3-5. 書き出して確認します
1. あなたはファイルに書き出します（YouTube向け）。
2. あなたは最初の30秒だけ見て、誤字と音量を確認します。
3. あなたは必要ならテロップだけ直して再書き出しします。

## 2-4. 例（VOICEVOX用に言い換えます）
### 2-4-1. 例を示します
言い換え前：
> SAAはハマると沼。

言い換え後（読み上げ向き）：
> AWSの資格、エスエーエーは、やり方を間違えると時間を使いやすいです。

## 2-5. チェックリスト / 練習問題
### 2-5-1. 音声チェックリストを使います
- [ ] あなたは読み間違いしやすい単語の読みを直しています。
- [ ] あなたは不自然な早口になっていません。
- [ ] あなたは句読点で間を作れています。

### 2-5-2. 15分練習をします
1. あなたは台本を60〜90秒分だけ用意します。
2. あなたはVOICEVOXで音声を書き出します。
3. あなたはiMovieで背景＋要点テロップ3枚だけ作ります。
4. あなたは書き出して「聞き取りやすさ」を1つだけ改善します。

## 2-6. まとめ
あなたはVOICEVOX→iMovieの流れにすると、**低スペMacでも安定しやすく**、短時間で1本を出せます。あなたは要点テロップに絞って、作業を軽くします。
